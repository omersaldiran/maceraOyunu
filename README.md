## Oynanış
Oyun komut sistemi üzerinden oynanmaktadır.
Oyunda 3 farklı karakter, 3 farklı silah ve 3 farklı zırh bulunmaktadır.
Oyun başında bir karakter belirlenir.
Çeşitli savaş lokasyonlarından geçilerek para ve özel koleksiyon itemleri toplanır.
Tüm lokasyonlardaki koleksiyon itemleri toplandığında, oyun kazanılır.
Karakterin Sağlık değeri sıfıra inerse, oyuncu yenilmiş olur.

İyi oyunlar !
